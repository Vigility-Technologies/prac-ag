# GEM Bid Management - Server

Backend server for GEM Bid Management System.

## Setup

### Install dependencies

```bash
npm install
```

### Copy `.env.example` to `.env` and fill in your credentials

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
npm start
```
